# Jobs Plug

A lightweight WordPress plugin for managing job postings and notices.

## Description

Jobs Plug is a simple, dependency-free WordPress plugin designed to help you manage job postings on your WordPress site.

## Installation

1. Upload the `jobs-plug` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings as needed

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher

## Features

- Lightweight and fast
- No external dependencies
- Properly namespaced to avoid conflicts
- Translation ready

## Development

The plugin uses the `JobsPlug` namespace to avoid conflicts with other plugins and themes.
